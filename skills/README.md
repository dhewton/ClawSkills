# Skills Index

Welcome to the growing skill collection inside **ClawSkills**.

This directory is where DannyBoy’s operator-grade skills live as they move from:
- concept
- to internal proving ground
- to hardened implementation
- to polished public release

## What qualifies as a ClawSkill

A skill belongs here when it aims to be:

- safe by default
- secure in how it handles secrets and runtime state
- performance-conscious
- reviewable by another operator
- useful in the real world
- stable enough to be trusted under pressure

## Release standard

Not every skill here is automatically public-ready.

Some entries are:
- active proving-ground work
- release candidates
- internal references awaiting extraction and cleanup

A skill earns public release when DannyBoy is satisfied that it is:
- understandable
- operationally clean
- low-drift
- safe to inspect
- worth attaching his name to
